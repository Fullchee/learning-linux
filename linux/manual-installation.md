### .ssh keys
1. Copy the ~/.ssh keys from your computer to the new computer
2. Mae sure to change the permissions for the private and public keys:
```
chmod 600 ~/.ssh/id_rsa
chmod 644 ~/.ssh/id_rsa.pub
```
